Derrière le projet Gedcom to JS

Ce projet était conçu initialement pour nos propres besoin mais n'hésitez pas à participer à nos côtés
Une idée fantastique ? un bug merveilleux ? Nous serions ravis de vous écouter

  • Comment ça marche?

    Construit en Javascript, ce projet permet à votre ordinateur de faire le travail tout seul. Nous sommes aidé pour cela par svj.js et svg2pdf.js.

  • Où partent mes données personnelles?

    Jamais plus loin que votre propre ordinateur. Merci Javascript :)

  • Comment participer?

    Vous pouvez créer des tickets en cas de bugs ou si vous avez une nouvelle idée, Nous serions ravis d'en discuter avec vous
    Vous pouvez égalemenr proposer des modifications interne au code

  • Puis-je utiliser / partager / modifier cet outil ?

    Faites ce que vous souhaitez tant que vous ne le conservez la gratuité et vous n'en prenez pas le crédit.
